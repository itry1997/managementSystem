# 用户权限管理系统
此仓库是后端代码，前端代码在vue-admin-template-4.4.0

## 功能包括 
用户、用户所处角色，以及角色拥有权限的增删改查

给用户分配不同角色

根据登录用户的角色不同（不同角色拥有不同权限）进行动态路由
